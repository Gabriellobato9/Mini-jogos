# SOBRE OS JOGOS 🕹️

Jogo de Perguntas e Respostas 🧠
O que é: Um quiz interativo focado em testar os conhecimentos do jogador sobre lógica de programação e a linguagem C.
Como funciona: O sistema apresenta 5 perguntas de múltipla escolha (com 4 alternativas cada). O jogador escolhe sua resposta e recebe um feedback imediato (correto ou incorreto) informando a resposta exata caso tenha errado.

Cobra na Caixa! 🐍
O que é: Um jogo de sorte e suspense para dois jogadores, inspirado no estilo "roleta russa", mas ambientado em uma tumba egípcia.
Como funciona: Existem 5 caixas fechadas. Atrás de uma delas está o botão que abre a saída (Vitória), e atrás de outra está uma cobra mortal (Derrota). O sistema sorteia quem começa e, a cada turno, os jogadores se revezam escolhendo uma caixa. Se a caixa estiver vazia, o turno passa.

Gousmas War (Guerra das Gosmas) ⚔️
O que é: Um jogo de estratégia matemática e duelo por turnos para dois jogadores.
Como funciona: Cada jogador começa com duas "Gosmas", cada uma com 1 ponto de vida. No seu turno, o jogador pode usar uma de suas gosmas para Atacar (somando o valor da sua gosma ao valor da gosma inimiga) ou Dividir/Redistribuir os pontos entre as suas próprias gosmas.
A Regra de Ouro: Se uma gosma inimiga ultrapassar 5 pontos, ela "estoura" e morre (vai a zero). O objetivo é forçar as gosmas do adversário a passarem de 5 pontos. O último jogador com gosmas vivas no tabuleiro vence.
